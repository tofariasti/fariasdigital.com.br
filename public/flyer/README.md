## Gerar PDF para gráfica

1. Abra `index.html` no Chrome
2. `Ctrl+P` → Destino: **Salvar como PDF**
3. Tamanho do papel: **A6** (105 × 148 mm)
4. Margens: **Nenhuma**
5. Ativar **Gráficos de segundo plano**
6. Salvar — enviar à gráfica

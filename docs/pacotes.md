## Web — Landing Page

**Público:** negócios locais que querem presença rápida no Google com WhatsApp integrado.

**Investimento:** a partir de **R$ 300**

| Entregável | Detalhe |
|------------|---------|
| Landing page | 1 página focada em conversão (modelo do segmento) |
| Personalização | Marca, textos e WhatsApp do cliente |
| Responsivo | Celular e desktop |
| SEO básico | title, description, Open Graph |
| Ajustes | 1 rodada após entrega |

**Não inclui:** domínio, hospedagem anual, sessão de fotos, redação longa de textos.

**Prazo típico:** 5–10 dias úteis após aprovação do escopo e recebimento do material (logo, fotos, textos).

**Diferença vs Site Institucional:** landing = 1 página com modelo pronto; institucional = múltiplas seções e mais conteúdo.

## Web — Site Institucional

**Público:** comércio local, clínicas, prestadores de serviço, profissionais liberais.

**Investimento:** a partir de **R$ 1.490**

| Entregável | Detalhe |
|------------|---------|
| Site institucional | Múltiplas seções (serviços, sobre, contato, localização) |
| WhatsApp | Botão flutuante + formulário que monta mensagem |
| Responsivo | Celular e desktop |
| SEO básico | title, description, Open Graph |
| Ajustes | 1 rodada após entrega |

**Não inclui:** domínio, hospedagem anual, sessão de fotos, redação longa de textos.

**Prazo típico:** 7–14 dias úteis após aprovação do escopo e recebimento do material (logo, fotos, textos).

**Upsell:** domínio + hospedagem no 1º ano (~R$ 150–250), Google Meu Negócio, manutenção mensal.

## Web — Loja Essencial (E-commerce)

**Público:** e-commerce, lojas pequenas e varejo com catálogo limitado.

**Investimento:** a partir de **R$ 2.990**

| Entregável | Detalhe |
|------------|---------|
| Catálogo | Até 50 produtos com foto, nome, preço e descrição curta |
| Pedido | Checkout via WhatsApp (sem gateway integrado) |
| Institucional | Sobre, contato, localização |
| Responsivo + SEO | Igual pacotes anteriores |
| Ajustes | 1 rodada após entrega |

**Não inclui:** Mercado Pago, PagSeguro, ERP, estoque em tempo real.

**Prazo típico:** 14–21 dias úteis.

## Web — Sob medida

**Público:** e-commerce completo, automação de WhatsApp, integrações, painéis admin e sistemas.

**Investimento:** orçamento personalizado (referência interna: R$ 5.000+ conforme escopo).

Inclui levantamento, proposta por fases e contrato com marcos de pagamento. Escopo típico: pagamento online, fluxos de atendimento no WhatsApp, integrações (ERP, marketplaces, APIs) e manutenção.

## Drone — Drone Imóvel (Pacote B — zona 1)

**Público:** imobiliárias, corretores, incorporadoras.

**Investimento:** a partir de **R$ 650** (região metropolitana de POA)

| Entregável | Detalhe |
|------------|---------|
| Fotos aéreas | Alta resolução, seleção editada |
| Vídeo | ~45 s editado, formato vertical e horizontal se combinado |
| Entrega | Arquivos para anúncio, portais e redes |
| Legal | Planejamento conforme legislação; SARPAS quando obrigatório |
| NF | Nota fiscal MEI |

**Adicionais:** zona de deslocamento, SARPAS, urgência, pacotes múltiplos imóveis — ver [precificação TechDrone360](../../techdrone360/leads/precificacao-servicos.md).

**Prazo típico:** fotos em poucos dias úteis; vídeo conforme combinado (3–10 dias úteis).

## Regras comerciais

1. **Sempre** enviar escopo por escrito no WhatsApp ou PDF antes de iniciar.
2. **Sinal de 50%** para projetos web acima de R$ 2.000 (opcional no primeiro cliente piloto).
3. **Não** prometer prazo sem validar material do cliente (fotos, textos, acesso).
4. Modelos do portfólio usam **marcas fictícias** — deixar claro que não são clientes pagantes. Após contratação, o site sai com a identidade do cliente.
